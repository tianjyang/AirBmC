### API Endpoints
** Root **
Root: Redirects to static homepage

** Cars Api **  
* `GET /api/cars/popular`  
* `GET /api/cars/byLocation  `


** Sessions Api **  
* `GET /api/session  `
* `POST /api/session  `
* `DELETE /api/session  `


** Comments Api **  
* `GET /api/comments  `
* `POST /api/comments  `
* `DELETE /api/comments  `
* `PATCH /api/comments  `

** Listings Api **  
* `GET /api/listings  `
